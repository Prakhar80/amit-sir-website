## Image Requirements

- Format: JPG or PNG
- Recommended dimensions: 800x600 pixels or higher
- Aspect ratio: 4:3 or original certificate aspect ratio
- File size: Optimized for web (under 500KB per image)

## Current Status

The StudentCertifications component is designed to display these certificates in an attractive animated slider.
The component will gracefully handle missing images with placeholder graphics showing the company logos and certification details.

## How to Add Images

1. Get the certificate images from the respective PDF files
2. Crop and optimize them for web display
3. Save them with the exact filenames listed above
4. Place them in this `/public/images/` directory
5. The website will automatically display them in the certifications section

## Note

The certificate images should maintain student privacy while showcasing their achievements.
Ensure you have permission to display student certificates publicly.
